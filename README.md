# Java Student Management System

Console-based student management application built using Java. This project focuses on object-oriented programming, constructors, ArrayLists, loops, user input, and CRUD operations.

## Features

* Add students
* View all students
* Search students by name
* Update student information
* Delete students
* Menu-driven user interface
* Store student records using ArrayList
* Constructor-based object creation
* Student not found handling

## Technologies

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner

## Run

Compile the Java files:

```bash
javac main.java Student.java
```

Run the program:

```bash
java main
```
